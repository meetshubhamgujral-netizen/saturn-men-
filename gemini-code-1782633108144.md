# Saturn – AI Market Understanding Dashboard 🪐

Saturn is a production-ready, high-fidelity business intelligence and market understanding analytics suite built completely on top of Python and Streamlit. This application bypasses complex multi-file configurations by integrating advanced descriptive stats, diagnostic hypothesis testers, adaptive machine learning model pipelines, and a deterministic conversational chatbot interface directly inside a clean, highly optimized **two-file monolithic architecture**.

The engine computes metrics, models feature relationships, and answers questions **exclusively over your uploaded data sets**, preserving perfect data integrity with zero third-party leakage.

---

## ⚡ Core Architecture Design

To keep project tracking and cloud deployment dead simple, the code has been optimized into exactly two functional core files:
1. **`requirements.txt`**: Manages exact library environments and scientific computation dependencies.
2. **`app.py`**: Houses the custom UI styling overrides, interactive tabs, processing algorithms, and chatbot framework.

---

## 🛠️ System Features & Capabilities

* **🏠 Home & Lander Portal**: Premium dark/light corporate SaaS landing hero banner detailing runtime health and node metrics.
* **📁 Ingest & Inspect Node**: Direct upload pipeline for raw files (`CSV`, `XLSX`, `JSON`) featuring fully automated type-inference engines that extract continuous vectors, dates, categorical blocks, and recommend optimal downstream target models.
* **🧹 Data Preprocessing Pipelines**: On-the-fly mutations including row-deduplication controls and missing coordinate handling frameworks (Mean/Mode or strict structural elimination).
* **📊 High-Fidelity Descriptive Suite**: Automated linear correlation heatmaps, high-density distribution histograms, and customizable cross-tabulation enterprise pivot engines.
* **📈 Diagnostic Analysis**: Automated inferential statistics running One-Way ANOVA tests over chosen category-to-continuous combinations with written clear plain-English business interpretation explanations.
* **🤖 Machine Learning Prototyper**: Self-configuring ML environment that detects target cardinality to deploy an internal **Random Forest Classifier** or **Regressor** with complete performance evaluation logs and explainable AI feature importance ranking.
* **💬 Deterministic Chatbot**: Bounded contextual analytics assistant engineered to pull information explicitly out of your loaded data table safely avoiding modern LLM hallucinations.
